Lab HTML

Tag Attribute การใช้งาน
h global attributes เช่น id,class,style ใช้เพื่อกำหนดส่วนหัว HTML
p global attributes เช่น id,class,style ใช้กำหนดย่อหน้า
div global attributes เช่น id,class,style ใช้เป็นที่เก็บขององค์ประกอบ HTML ในรูปเเบบ Block
img global attributes,alt,height,width,src,sizes,ismap,longdesc ใช้สำหรับใส่รูปภาพในหน้า HTML
a global attributes,href,rel,type,target ใช้กำหนดการเชื่อมโยงใช้เพื่อเชื่อมโยงจากหน้าหนึ่งไปอีกหน้าหนึ่ง
ul global attributes เช่น id,class,style ใช้กำหนดรายการที่ไม่มีลำดับเป็นสัญลักษณ์แสดงหัวข้อย่อย
ol global attributes,reversed,start,type ใช้กำหนดรายการเป็นตัวเลขหรือตัวอักษร
table global attributes เช่น id,class,style ใช้กำหนดตาราง HTML
form global attributes,action,autocomplete,enctype,method,name,rel,novalidate,target ใช้สำหรับสร้างฟอร์ม HTML สำหรับการป้อนข้อมูลของผู้ใช้
button global attributes,name,type,value,autofocus,disabled,form ใช้กำหนดปุ่มที่คลิกได้
Tag ความเหมือน ความต่าง
hr & br เว้นบรรทัด br จะเว้นบรรทัดปกติ
hr จะเว้นบรรทัดเเล้วมีเส้นระหว่างบรรทัดให้ด้วย
ul & ol กำหนด list ต่างๆ ul จะเป็นลัญลักษณ์ข้างหน้า list
ol จะเป็นตัวเลขข้างหน้า list
select & checkbox & radio button ใช้เป็ยตัวเลือกใน from select จะเป็นกล่องเดียวพอกดลูกศรที่มุมขวาจะมีตัวเลือกสไลด์ลงมาเพิ่ม
checkbox จะเป็นกล่องสี่เหลี่ยมเล็กๆหน้าข้อความไว้ให้เลือกเพียงเเค่ 1 ตัว
radio button จะเป็นวงกลมเล็กๆหน้าข้อความไว้ให้เลือกได้หลายตัว
div & span เป็นคอนเทนเนอร์เป็นที่เก็บขององค์ประกอบต่างๆ div จะเป็นของ Block level elements
span จะเป็นของ lnline level elements
script & style ใช้กำหนดข้อมูลคำสั่ง script กำหนดข้อมูลคำสั่งของ css
style กำหนดข้อมูลคำสั่งของ java script
input type การใช้งาน
button เป็นปุ่มกด
checkbox กำหนดปุ่มตัวเลือกให้ผู้ใช้เลือกได้1ตัวหรือมากกว่า
color ปุ่มไว้ใช้ในการเลือกสี
date ปุ่มไว้ใช้ในการเลือกวันเดือนปี
datetime-local ระบุข้อมูลของช่วงเวลา
email ใช้สำหรับใส่ข้อมูลอีเมลล์
file ใช้ในการใส่ไฟล์ลงไป
hidden ใช้ซ้อนข้อมูลให้มองไม่เห็น
image ใช้ในการเเสดงภาพ
month ใช้ในการเลือกเดือนเเละปี
number กำหนดฟิลด์อินพุตเป็นตัวเลขเเละตั้งข้อกำหนดได้
password ใช้ในการใส่ password โดยจะไม่เเสดงเป็นเลข
radio กำหนดปุ่มตัวเลือกให้ผู้ใช้เลือกหนึ่งตัวเลือก
range ป้อนขอมูลตัวเลขในรูปแบบเเถบเลื่อน
reset เซ็ตค่าแบบฟอร์มทั้งหมดเป็นค่าเริ่มต้น
search ช่องที่ใช้ในการป้อนข้อมูลที่จะค้นหา
submit ปุ่มกดยืนยัน
tel ใช้สำหรับinputที่เป็นหมายเลขโทรศัพท์
text ใช้ใส่ข้อมูลตัวอักษร
time ใช้ในการเลือกเวลา
url ใช้ใส่ข้อมูล web address
week ใช้ในการเลือกสัปดาห์และปี
Tag ผลลัพธ์ คำอธิบาย
strong Ter ทำให้ตัวอักษรหนา
label ใช้กำหนดป้ายชื่อให้กับ ช่องรับข้อมูลอยู่ที่หน้าform
del Ter ทำให้มีขีดทับที่กลางตัวอักษร
b Ter ทำให้ตัวอักษรหนา
em Ter ทำให้ตัวอักษรเอียง
mark Ter ไฮไลท์สีที่ตัวอักษร
i Ter ทำให้ตัวอักษรเอียง
sub Ter ตัวอักษรจะห้อยข้างล้าง
sup Ter ตัวอักษรจะห้อยข้างบน
